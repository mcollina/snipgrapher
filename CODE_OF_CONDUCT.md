# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus on technical content, not personal attacks.
- Accept feedback gracefully.

## Unacceptable Behavior

- Harassment, threats, discrimination, or hateful language.
- Personal attacks or deliberate intimidation.
- Publishing private information without consent.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and issues that violate this code of conduct.

## Reporting

Please report incidents to the maintainers via private channel (GitHub private contact or direct message).
